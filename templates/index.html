Atención: Git LFS Requerido

Los modelos de procesamiento de lenguaje natural locales no se han podido cargar correctamente. Esto suele ocurrir porque el repositorio se clonó sin activar Git LFS, lo que hace que los archivos de pesos sean solo punteros de texto.

Para solucionarlo, instala Git LFS, ejecuta en tu terminal: git lfs install y git lfs pull, y luego reinicia esta aplicación.
